## 함수의 기초

- 함수의 정의 - 특정한 기능을 하는 코드의 조각

- 매번 다시 작성x, 호출하여 간편히 사용

- Error = more code

- ☆ 사용자 함수

  ```python
  def function_name(parameter):
  	code block
  	return returning_value
  	
  function_name(Argument)
  ```

  | 정의      | 호출     |
  | --------- | -------- |
  | parameter | Argument |
  | 매개변수  | 전달인자 |
  | 인수      | 인자     |
